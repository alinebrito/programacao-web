```html
<!DOCTYPE html>
<html>
<head>
<style>
body {
  font-family: Arial;
  background-color: #e6f0ff;
  text-align: center;
}

.container {
  background-color: white;
  width: 300px;
  margin: 50px auto;
  padding: 20px;
  border-radius: 10px;
}

  color: #3366cc;
}

  font-size: 20px;
}

  background-color: #3366cc;

  border-radius: 8px;
}
</style>

<body>

  <h1>Minha primeira página</h1>
  <p>Esse é um parágrafo de exemplo.</p>
  <button>Começar</button>
</div>

</body>
</html>
```
